# Stock in Ounces

Track stock prices priced in Gold Ounces. Visualize the true value of assets by stripping away currency inflation.

## [Live Demo](https://emunozgutier.github.io/Stock-In-Ounces/)

Check out the live application here: **[https://emunozgutier.github.io/Stock-In-Ounces/](https://emunozgutier.github.io/Stock-In-Ounces/)**

## Features

- **Price in Gold**: View stock and crypto prices divided by the price of Gold.
- **Trendlines**: Analyze long-term trends with Linear and Logarithmic regression.
- **Rainbow Chart**: Visualize valuation "bands" (Top/Bottom 10% and 20%) to identify potential overbought/oversold conditions.
- **Time Ranges**: 1Y, 5Y, 10Y, and Max history.

## Development
